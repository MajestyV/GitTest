## III. Common command lines (常用命令行)

### A. git

```git push```命令用于从将本地的分支版本上传到远程并合并。

```
$ git push <远程主机名> <本地分支名>:<远程分支名>
```

如果本地分支名与远程分支名相同，则可以省略冒号:

```
$ git push <远程主机名> <本地分支名>
```
